Git - the ultimate VCS

Version control schemes: Local, Centralized, Distributed

Schemas courtesy of Scott Chacon - "Pro Git" - 2009

Local Version Control

  • Record stored on host
  • Error prone
  • rcs on MacOS X

Centralized Version Control

  • Record stored on a server
  • Version Control available only when network is up
  • CVS, SVN

Distributed Version Control

  • Record stored on a each local copy
  • Share the changes between repositories
  • Version Control is available anywhere, anytime
  • Bazaar, Mercurial, Git

Snapshots vs Diffs

  • Old VCS
  • Git

Basic commands

git init

  • Initiate empty repository
  • Use at the top of the folder hierarchy to version
jopasserat@vapula:/tmp$ cd git_tutorial/ex1/
jopasserat@vapula:/tmp/git_tutorial/ex1$ git init
Initialized empty Git repository in /private/tmp/git_tutorial/ex1/.git/

git add

  • Start tracking a file
  • Takes files or whole folders as parameter
jopasserat@vapula:/tmp/git_tutorial/ex1$ ls
bar	foo	foobar
jopasserat@vapula:/tmp/git_tutorial/ex1$ git add bar
jopasserat@vapula:/tmp/git_tutorial/ex1$ git add .

git status

  • Display the current status of the working tree (what is to be commited, what is staged for commit...)
jopasserat@vapula:/tmp/git_tutorial/ex1$ git status
# On branch master
#
# Initial commit
#
# Changes to be committed:
#   (use "git rm --cached ..." to unstage)
#
#	new file:   bar
#	new file:   foo
#	new file:   foobar
#

git commit

  • One of the most important command!
  • Creates a snapshot of each file staged for commit
jopasserat@vapula:/tmp/git_tutorial/ex1$ git commit bar -m 'First commit of bar file'
[master (root-commit) 6ba7d15] First commit of bar file
0 files changed
create mode 100644 bar
jopasserat@vapula:/tmp/git_tutorial/ex1$ git commit -am 'Commit the remaining files'
[master cd2c39c] Commit the remaining files
0 files changed
create mode 100644 foo
create mode 100644 foobar

git log

  • Browse the commits history
  • Displayed using /usr/bin/less on UNIX (same commands as Vi...)
jopasserat@vapula:/tmp/git_tutorial/ex1$ git log
commit cd2c39cc6a6bb6a08f3d74c37ecc0dfd71e5d604
Author: Jonathan Passerat-Palmbach

    First commit of bar file

git diff

  • Show the changes between the working tree version of a file and a commited version
  • + for inserted lines
  • - for removed lines
jopasserat@vapula:/tmp/git_tutorial/ex1$ vi bar
jopasserat@vapula:/tmp/git_tutorial/ex1$ git diff bar
diff --git a/bar b/bar
index e69de29..270c611 100644
--- a/bar
+++ b/bar
@@ -0,0 +1 @@
+hello, world!

git mv

  • Rename tracked file
  • Move tracked file from one folder to another
jopasserat@vapula:/tmp/git_tutorial/ex1$ git mv foo foo2
jopasserat@vapula:/tmp/git_tutorial/ex1$ mkdir dir
jopasserat@vapula:/tmp/git_tutorial/ex1$ git mv bar dir/
jopasserat@vapula:/tmp/git_tutorial/ex1$ git ls-files
foo2
dir/bar
foobar

git rm

  • Remove file from disk
  • Stop tracking a file
jopasserat@vapula:/tmp/git_tutorial/ex1$ git rm foo2
rm 'foo2'
jopasserat@vapula:/tmp/git_tutorial/ex1$ ls
foobar	dir
jopasserat@vapula:/tmp/git_tutorial/ex1$ git rm --cached dir/bar
rm 'dir/bar'
jopasserat@vapula:/tmp/git_tutorial/ex1$ ls
foobar	dir
jopasserat@vapula:/tmp/git_tutorial/ex1$ git ls-files
foobar
